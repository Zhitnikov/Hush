localtunnel (loca.lt)

Не вводите свой IP (например 178.66.130.5).

На странице loca.lt показан IP сервера туннеля (часто 45.x.x.x и т.п.) — в поле нужно ввести именно его, один в один.

Если ввести свой домашний IP — форма не пустит, появится ошибка в Chrome.

Удобные варианты

  • Телефон в той же Wi‑Fi: https://<IP-ПК>:3443 (без туннеля).
  • ПК, без loca.lt: http://localhost:3333
  • Телефон в Wi‑Fi: https://<IP-ПК>:3443
  • Из интернета: https://….loca.lt после npm run dev:public (preview, не dev — меньше 502)
  • ПК без формы IP: /__hush_lt/
  • Без loca.lt (ssh): set HUSH_TUNNEL=run + npm run dev:public
  • Только HTTP (не для E2E): set HUSH_TUNNEL=bore.
